## Description

Backend repository du projet de pokedex

## Installation

```bash
$ curl https://get.volta.sh | bash #installation de Volta
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
